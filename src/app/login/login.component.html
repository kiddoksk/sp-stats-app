Students Stats Dashboard

Features:

You can view the statistics from the student data which is populated. The data consists of various fields using which you can filter and visualize the data as Graphs Steps to use the application

1. In the Dashboard, you will see a list of field where you can enable or disable the toggle to filter the data

2. You can apply filters all the available fields and save the form the visualize the data in graphs to get more insights

3. You can try various combinations and the graphs will be populated based on the filters and combinations you select
